# Code of Conduct

We commit to a respectful, harassment-free community.

## Expected
- Be respectful and professional.
- Accept constructive feedback.
- Assume good intent.

## Unacceptable
Harassment, discrimination, or personal attacks.

## Reporting
Report incidents privately via Security Advisories or repo Issues to the maintainers. Maintainers may take any action they deem appropriate, including warnings or bans.
